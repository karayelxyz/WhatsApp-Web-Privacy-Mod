# WhatsApp Web Privacy Mod
Privacy Mode for WhatsApp Web made with UserCSS
</div>


### 🔒 WhatsApp Web – Stay Private in Public



#### What it does:

- 📃 Blurs the **chat list** (left sidebar).
- 🖼️ Blurs **images** in messages.
- 📹 Blurs **videos**.
- 🎞️ Blurs **GIFs**.
- 💬 Blurs **stickers**.
- 🔗 Blurs **link preview images**.
- 📍 Blurs **shared location**.
- 📞 Blurs **call logs**
- 👤 Blurs **images and videos** in contact info.
- ☎️ Blurs **phone numbers** in contact info.

Hovering over any blurred content will reveal it.
